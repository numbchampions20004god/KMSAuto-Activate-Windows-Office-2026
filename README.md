## ⚙️ How to Use

### For Windows Activation:
1. Run `KMSAuto.exe` as Administrator
2. Click on **"Activate Windows"**
3. Wait for the process to complete
4. Restart your computer
5. Check activation status in Settings → Update & Security → Activation

### For Office Activation:
1. Run `KMSAuto.exe` as Administrator
2. Click on **"Activate Office"**
3. Wait for the process to complete
4. Restart all Office applications
5. Check activation in File → Account

### Additional Options:

## ❓ Troubleshooting

| Problem | Solution |
|---------|----------|
| "Not recognized as internal command" | Run as Administrator (right-click → Run as Administrator) |
| Activation fails | Check your internet connection. Try a different KMS server |
| Windows already activated | No need to run. Check your activation status first |
| Office not detected | Make sure Office is installed and was launched at least once |
| "Missing MSVCRT.dll" | Install Visual C++ Redistributable packages |
